# Contributing to NexHeal

1. **Fork the Repository**: [https://github.com/shubhamkumar-git01/NexHeal](https://github.com/shubhamkumar-git01/NexHeal) and clone it locally.
2. **Create a Feature Branch** (`git checkout -b feature/amazing-feature`).
3. **Commit your Changes** using semantic commit messages (`feat:`, `fix:`, `docs:`).
4. **Push to the Branch** (`git push origin feature/amazing-feature`).
5. **Open a Pull Request** against the `main` branch on GitHub.

Please ensure all tests pass (`npm test`) and code passes ESLint validation before submitting a PR.
