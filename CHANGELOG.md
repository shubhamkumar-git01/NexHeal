# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0-rc.1] - 2026-07-07

### Added
- **Global Command Center**: Centralized BI analytics with Recharts integration.
- **AI Gateway**: Provider-agnostic AI adapters for generative medical insights.
- **Hospital ERP**: Ward, ICU, Staff, and Pharmacy inventory management.
- **Emergency Dispatch**: Real-time ambulance tracking via WebSockets.
- **Community Assistance**: Volunteer verification, points, and NGO integrations.
- **Electronic Health Records (EHR)**: Full longitudinal patient history, vitals, prescriptions.
- **Enterprise Infrastructure**: Stubs and architecture for Redis caching, background job queues, and robust logging.
- **Security**: Strict CSP via Helmet, rate-limiting, and RBAC implementation.
- **DevOps**: Multi-stage Dockerfiles and Docker Compose orchestration.
- **Testing**: Jest and React Testing Library scaffolding.
